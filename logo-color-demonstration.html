Accessibility Features

  • WCAG AA Compliance: All text meets minimum 4.5:1 contrast ratio
  • Large Text Support: 3:1 ratio maintained for headings and large elements
  • Color Independence: Information not conveyed by color alone
  • High Contrast Mode: Automatic adaptation for accessibility preferences
  • Screen Reader Friendly: Semantic color usage with proper ARIA labels
